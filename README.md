<h1>書籍系統</h1>

<p>用於新增書籍到至MySQL server，並在另一個頁面上顯示書籍資訊及評論內容和新增評論</p>

<h2>環境</h2>

<p>IntelliJ</p>

<h2>安裝指南</h2>

<p>需安裝 Tomcat 及下載 mysql-connector 和 JSTL</p>
